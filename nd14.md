#### tanimoto係数を計算するメソッド

```ruby
# tanimoto係数メソッド(数値の配列だけの比較をする)

def tanimoto(a,b)
	na=a.count(1)						# aの１の数の合計
	nb=b.count(1)					# bの１の数の合計
	azb=a.zip(b) 					# aとbをzipで合成（[1,1] が共通集合）
	nab=azb.count([1,1])            # aとbの共通集合の要素の数
	return nab/(na+nb-nab).to_f		# tanimoto係数
end

```

## 配列の内容をいろいろな条件でフィルターする

mapメソッドのように配列を出力するイテレータ
